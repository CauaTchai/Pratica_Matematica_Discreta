# 📌 Projeto: Fundamentos de Computação

Este projeto consiste no desenvolvimento de três programas que abordam conceitos fundamentais de matemática computacional e algoritmos, com foco em aprendizado prático e visualização passo a passo dos processos.

## 🚀 Funcionalidades

### 1) Calculadora de Bases

* Conversão entre bases: **decimal, binária e hexadecimal**
* Exibição do **passo a passo** das conversões
* Operações entre dois números na mesma base:

  * Soma
  * Subtração (sem resultados negativos)
  * Multiplicação
* Caso a subtração resulte em número negativo, o programa informa que a operação não é suportada

### 2) Algoritmo Estendido de Euclides

* Cálculo do **MDC (Máximo Divisor Comum)** entre dois números inteiros

* Exibição do **passo a passo** do algoritmo

* Representação do resultado como combinação linear:

  `mdc(a, b) = s·a + t·b`

* Exibição das equações utilizadas no processo (sem necessidade de mostrar substituições intermediárias)

### 3) Crivo de Eratóstenes

* Geração de todos os **números primos menores que n**
* Implementação obrigatória do **Crivo de Eratóstenes**
* Exibição do **passo a passo** do processo de filtragem dos números

## 🛠️ Como Iniciar o Programa

### 1. Preparando o ambiente C# (Crivo)
No diretório onde está o arquivo "Crivo de Eratóstenes.cs", execute no terminal:
dotnet new console --force

### 2. Iniciando o servidor Flask (Backend)
No terminal, execute:
python app.py

### 3. Abrindo a Interface (Frontend)
Abra o arquivo "index.html" no seu navegador.

## ⚙️ Requisitos do Projeto

* Linguagem de programação livre
* Interface simples e amigável
* Implementação própria (sem uso de bibliotecas prontas para os algoritmos)
